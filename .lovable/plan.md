

# CBIT Connect — Complete Web App Plan

A comprehensive digital ecosystem for CBIT students, faculty, clubs, alumni, and placement activities. Built as a responsive web app with mock data, ready for backend integration later.

---

## Phase 1: Foundation & Navigation

- **App shell** with sidebar navigation and top bar
- **Role selector** (Student / Alumni / Faculty / Club Admin / College Admin) to simulate different user views
- **Responsive layout** that works great on mobile and desktop
- **Color scheme**: Modern, clean dashboard style with CBIT-inspired branding (blue/green tones)

---

## Phase 2: Home Dashboard

- Welcome greeting with user role context
- **Quick-access tiles** linking to CBIT website, LMS, Exam Cell, Library (external links)
- **Announcements feed** with today's highlights
- **Upcoming events** carousel
- **Placement alerts** banner
- **Club highlights** section

---

## Phase 3: Student Module

- **Timetable** — weekly class schedule view
- **Attendance** — subject-wise attendance percentage display
- **Internal marks** — marks table by subject
- **Exam notifications** — list of upcoming exams
- **Study materials** — downloadable resources organized by subject
- **Event registrations** — browse & register for events
- **Lost & Found** — post and browse lost items
- **Bus timings** — route-wise schedule

---

## Phase 4: Clubs & Communities

- **Club directory** with search/filter
- **Club profile pages** — description, team members, gallery, achievements
- **Upcoming events** per club
- **Join request button**

---

## Phase 5: Alumni Module & RFC System

- **Alumni directory** with filters (company, role, department, batch)
- **Alumni profile cards** — name, batch, company, LinkedIn, mentorship/referral availability
- **Request for Connect (RFC)** — students can send mentorship/referral/guidance requests
- **RFC inbox** — alumni can accept/reject with simulated in-app messaging
- **Notification indicators** for RFC status updates

---

## Phase 6: Placement & Internship Hub

- **Internship postings** — browse with filters
- **Referral request** section
- **Interview experiences** — community-submitted stories
- **Placement prep materials** — organized resources

---

## Phase 7: Faculty Module & Newsletters

- **Faculty directory** with department filter
- **Faculty profiles** — subjects, office hours, research papers
- **Campus news feed** — department newsletters, achievements, hackathons, fests, student startups

---

## Phase 8: Community Features

- **Doubt Forum** — post questions, senior answers, upvoting
- **Student Marketplace** — buy/sell books, cycles, calculators, campus ride sharing
- **Feedback & Polls** — anonymous feedback forms and polls for mess, campus, events

---

## Phase 9: Utility Features

- **Digital Student ID** with QR code display
- **Smart Notification Center** — centralized view of all alerts (exams, events, alumni responses, internships, club meetings)
- **Search** — global search across all modules

---

## Design Approach

- Clean, modern dashboard UI with card-based layouts
- Mobile-first responsive design
- Consistent icon system (Lucide icons)
- Tab-based navigation within modules
- All data will be realistic mock data, structured for easy backend replacement later
